# Array
Fixed size, homogeneous elements (same data type), contiguously located in memory.
Random Access (O(1), directly accessed) by using Index.
Insertion & Deletion is hard.

# Stack
insertions and deletions are made at one end, called top.
LIFO (Last-In First-Out) architecture.

ADT
create(capacity) return stack
push(stack, item)
pop(stack) return item
peek(stack) return top-item
isFull(stack) return true/false
isEmpty(stack)

# Queue
insert at tail(rear), delete from head(front).
FIFO (First-In First-Out) architecture.

ADT
queue create(capacity)
void enqueue(queue, item)
item dequeue(queue)
item front(queue)
item rear(queue)
bool isFull(queue)
bool isEmpty(queue)
