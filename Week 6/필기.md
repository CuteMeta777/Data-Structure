# 기본 개념 (용어 정리)

Node : 점
Edge : 간선
Parent & Child : 간선으로 이어진 두 Node 간의 상하 관계를 나타냄. 위쪽이 Parent, 아래쪽이 Child.
Siblings : 같은 부모를 지닌 Node들.

Root : 부모가 없는 Node.
Leaf : 자식이 없는 Node.
Internal : Root도 아니며, Leaf도 아니면 Internal이다.
Path : 특정 Node에서 다른 Node로 가는 경로를 뜻함. Tree에서는 항상 unique한 path만이 존재한다.
Ancestor : path를 거쳤을 때, 위에 있으면 Ancestor.
Descendant : path를 거쳤을 때, 아래에 있으면 Descendant.

Degree : ([Degree of Node]는 자식의 수를 의미) ([Degree of Tree]는 Tree내 Degree의 최댓값)
Level : ([Level of Node]는 root에서 해당 node로 향하는 path의 길이를 뜻함) ([Level of Tree]는 Tree내 Level의 최댓값)
Width : [Width of Tree]는 동일 Level에서의 Degree의 합의 최댓값을 의미.
Size : 전체 Node의 개수

# Binary Tree의 종류

Full : 
모든 노드가 0개 또는 2개의 자식을 가진다.

Complete : 
마지막 레벨을 제외한 모든 레벨에 노드가 꽉 차있다. 
마지막 레벨에는 왼쪽부터 노드가 채워진다.

Balanced : 
A binary tree where the heights of the left and right subtrees of every node differ by at most 1
참고로 Height란, 특정 노드에서 가장 밑바닥에 있는 Leaf 노드까지 내려가는 가장 긴 Path의 길이를 뜻함.

Perfect : 
모든 Internal 노드가 2개의 자식을 가진다.
모든 Leaf 노드가 같은 레벨에 존재한다.
